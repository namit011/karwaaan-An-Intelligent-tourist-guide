KARWAAN - The Intelligent Tourist Guide

“Karwaan” is a software which helps tourists with good tourist spots to visit. We know that in a huge country like India, there is a lot of geographical and ecological diversity. There are mountain ranges, deserts, rainforests, and as a tourist, it can get tough to get the hang of it. Enter, Karwaan. It is a software through which you can select your location or the location you want to visit and will show you what the attractions nearby are. These attractions are curated from a list of over 300 handpicked attractions for which ratings have been taken from a huge list of visitors. The system uses machine learning algorithms, data representation techniques and a vast database to provide user with the results.
